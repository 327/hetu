配置页面过程中, 经常遇到页面白屏或者菊花 loading, 90%的可能是接口错误, 可按照如下步骤排查错误。

##  接口异常

### 1. 检查网络请求

鼠标右键, 点击检查  
然后依次打开`Network XHR`面板, 查看接口响应状态码是否异常. `2xx 3xx`通常表示成功,`4xx 5xx`通常表示异常。
![](https://user-gold-cdn.xitu.io/2019/7/4/16bbb09224fb005e?w=918&h=302&f=png&s=64690)

### 2. 查看错误详情
