## 河图

JSON 生成页面工具, 内置 CMS 系统业务组件库

## 安装依赖

```
npm run _init
```

## 启动服务

```
npm start
```

## 打包 npm 包

```
npm run build
```

## 测试代码

```
npm run test
```

## 发布 npm 包

```
npm run publish
```

## 启动文档服务

```
npm run start:site
```

## 发布文档

```
npm run deploy:site
```
